## [HIG] APP Icon

> https://developer.apple.com/design/human-interface-guidelines/ios/icons-and-images/app-icon/
>
> 꼭 읽어보기 !

모든 앱에서 App Store에서 주목을 끄는 홈 화면에서 두르러지고 아름답고 기억에 남을만한 아이콘이 필요하다. 아이콘은 앱의 목적을 한눈에 알 수 있는 첫 번째 기회이다. 또한 설정 및 검색 결과와 같이 시스템 전체에 나타난다.

- **단순함을 갖아라.**

  앱의 본질을 파악하고 그 요소를 단순하고 독특한 모양으로 표현하는 단일 요소를 찾는다. 아이콘의 내용이나 모양이 지나치게 복잡하면 특히 작은 크기의 경우 세부 정보를 구분하기 어려울 수 있다.

- 단일 초점 지점을 제공하라.

  한 지점을 중심으로 즉각적인 주의를 끌며 앱을 명확하게 식별할 수 있는 아이콘을 디자인해야한다.

- 인식할 수 있는 아이콘을 디자인하라.

  사람들은 아이콘을 분석하여 아이콘이 무엇을 나타내는지 알아낼 필요가 없다. 예를 들어 메일 응용 프로그램 아이콘은 일반적으로 메일과 관련된 봉투를 사용한다. 시난을 내어 앱의 목적을 예술적으로 표현하는 아름답고 매력적인 추상적인 아이콘을 디자인해야한다.

- **배경을 간단하게 유지하고 투명성을 피해라.**

  아이콘이 불투명한지 확인하고 배경을 혼란스럽게 하지 말아야 한다. 간단한 배경으로 사용하면 근처에 있는 다른 앱 아이콘에 과부하가 걸리지 않는다. 전체 아이콘을 내용으로 패울필요는 없다.

- **로고가 필수적이거나 로고의 일부일 때만 단어를 사용해라.(단어 사용X)**

  앱의 이름이 홈 화면의 아이콘 아래에 나타난다. 이름을 반복하는 필수적이지 않은 단어를 포함시키거나 'Watch'또는 'Play'와 같은 앱으로 할 일을 사람들에게 알리지 말아야 한다. 디자인에 텍스트가 포함되어 있는 경우 앱에서 제공하는 실제 콘텐츠와 관련된 단어를 강조해야한다.

- 사진, 스크린 샷 또는 인터페이스 요소를 포함하지 말아라.

  사진의 세부 사항은 작은 크기에서 보기가 매우 어려울 수 있다. 스크린 샹은 앱 아이콘에 비해 너무 복잡하며 일반적으로 앱의 목적을 전달하는데 도움이 되지 않는다. 아이콘의 인터페이스 요소는 오해의 소지가 있으며 혼동을 준다.

- Apple하드웨어 제품의 복제품을 사용하지 말아라.

  Apple제품은 저작권이 있으며 귀하의 아이콘이나 이미지로 복제할 수 없다. 일반적으로 하드웨어 디자인은 자주 변경되는 경향이 있으므로 아이콘 복제본을 표시할 수 있으므로 장치의 복제본은 표시하지 말아야한다.

- **인터페이스 전체에 앱 아이콘을 두지 말아라.**

  앱 전체에서 다양한 용도로 사용되는 아이콘을 보는 것은 혼란스러울 수 있다. 대신 아이콘 색 구성표를 통합하는 것을 고려해야한다. [색상](https://developer.apple.com/design/human-interface-guidelines/ios/visual-design/color/)

- 다른 배경 화면에 대한 귀한의 아이콘을 테스트해라.

  사람들이 홈 화면으로 선택한 배경 화면을 예측할 수 없으므로, 앱을 밝거나 어두운 색상으로 테스트하지 말아야한다. 장치가 움직일 때 원근감이 바뀌는 동적 배경을 가진 실제 장치에서 사용해봐야 한다.

- **아이콘 모퉁이를 사각형으로 유지해라.**

  아이콘 모서리를 자동으로 반올림하는 마스크가 적용된다.

- **앱 아이콘 속성을 준수해라.**

  | 속성    | 값                                   |
  | ------- | ------------------------------------ |
  | 체재    | PNG                                  |
  | 색 공간 | sRGB또는 P3(색상 관리 참조)          |
  | 레이어  | 투명도가 없는 평면                   |
  | 해결    | 다양하다. 이미지 크기 및 해상도 참조 |
  | 모양    | 둥근 모서리가 없는 정사각형          |

- **다른 장치에 대해 다른 크기의  아이콘을 제공하라.**

  지원하는 모든 기기에서 앱 아이콘이 잘 보이는지 확인해야한다.

- 설정 아이콘에 오버레이 똔느 경계선을 추가하지 마라.

  iOS는 모든 아이콘에 1픽셀 획을 자동으로 추가하여 설정의 흰색 배경에 잘 보이게 한다.
